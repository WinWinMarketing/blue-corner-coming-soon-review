# Blue Corner coming-soon — concept goals

## Shared goal

Replace the generic coming-soon page with a credible, emotionally direct first encounter for Canadian men considering support. Every route uses the exact same supplied masthead, measured two-column hero, clean 3:2 boxing-ring image, locked core copy, section order, single local-only member form, crisis access, responsive layout, and official palette: blue `#197CE3`, navy `#042874`, yellow `#EFC82C`, and off-white `#EFF3F9`.

## 01 — Reference master

Route: `04-first-bell`.

This is the screenshot/reference implementation. It has no concept add-on and is the immutable baseline for every comparison.

## 02 — No perfect words needed

Route: `03-open-corner`.

Add exactly one practical module with three opening lines a man can use to begin a conversation. Nothing else differs from the master.

## 03 — The first ten minutes

Route: `06-kitchen-light`.

Add exactly one expectation-setting module that makes a first session easier to picture without scripting or overpromising it. Nothing else differs from the master.

## 04 — The Corner Standard

Route: `09-listening-room`.

Add exactly one module that makes therapist screening, consent, scope, and fit criteria visible. It does not add a second form. Nothing else differs from the master.

## 05 — Between-round plan

Route: `11-on-the-other-end`.

Add exactly one module describing what a private take-home next-step plan could contain without clinical or timing promises. Nothing else differs from the master.

## 06 — Bring someone into your corner

Route: `12-two-ways-in`.

Add exactly one consent-led module for optionally involving a trusted person. Nothing else differs from the master.

## 07 — Private readiness check

Route: `01-ringside-signal`.

Add exactly one private reflection module with no score, ranking, result, or diagnosis. Nothing else differs from the master.

## 08 — Between-round reset

Route: `02-between-rounds`.

Add exactly one short grounding card that clearly states it is not therapy, medical advice, or a substitute for crisis support. Nothing else differs from the master.

## 09 — Ask for support

Route: `05-someone-in-your-corner`.

Add exactly one local-only builder for a specific support request. Nothing sends from the page. Nothing else differs from the master.

## 10 — Workday boundary

Route: `07-workday-pause`.

Add exactly one workplace boundary drafting aid with clear legal, medical, and policy limits. Nothing else differs from the master.

## 11 — Body-signal check-in

Route: `08-what-hands-say`.

## 12 — Questions before booking

Route: `10-blue-margin`.

Add exactly one selectable booking-question checklist with local clipboard copy and a manual fallback. Nothing else differs from the master.

## Review rule

Judge every route against ordinal 01. A concept succeeds only if its single labeled module is useful and the reference page around it remains unchanged.

## Editorial gate

The `85%`, `#1`, and `~300%` statements are retained verbatim from the supplied reference page for design review. Verify each claim and link it to a primary source before any production launch.
